# Host Web Vercel

## Jalankan lokal

```bash
npm install
npm run dev
```

## Deploy ke Vercel

1. Upload folder ini ke GitHub.
2. Import repository ke Vercel.
3. Deploy.

## Route

- `/` -> pilihan Android / iOS
- `/android` -> tampilan host Android
- `/ios` -> tampilan host iOS
- `/api/android` -> output text/plain Android
- `/api/ios` -> output text/plain iOS
